# 프로젝트 개요

- 제목: <쿠키런:오븐브레이크> 모작 (제작사: 데브시스터즈)
- 장르: 사이드 스크롤 러닝 액션 게임
- 개발기간: 3주
- 개발도구: Unity
- 버전: 2022.3.2f1
- 개발언어: c#
- 플랫폼: 모바일
- 팀원: 4명
- 모작 목표:
  - 모바일 맞춤 UGUI 최적화 게임의 구현
  - State Pattern(디자인패턴) 구현
    - '쿠키런 캐릭터'의 다양한 상태 전이(달리기, 점프, 슬라이드, 보너스 타임 등) 로직의 단순화
    - 쿠키별 상태 추가의 확장성 부여
